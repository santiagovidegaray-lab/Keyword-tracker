👋 Configura tu tracker

Solo necesitas un repositorio de GitHub gratuito. Es donde se guardarán tus datos de keywords cada vez que importes un CSV nuevo.

1
Crea un repositorio en GitHub
Ve a github.com/new, ponle nombre (ej: aso-tracker), márcalo como público y haz clic en "Create repository".
2
Activa GitHub Pages
En tu repo → Settings → Pages → Source: Deploy from branch → Branch: main → Save. En unos minutos tendrás tu URL.
3
Crea un token de acceso
Ve a github.com/settings/tokens/new → Nota: "ASO Tracker" → Expiration: No expiration → Scope: ✅ repo → Generate token. Cópialo.
Se guarda solo en este navegador. Nunca sale de tu dispositivo.
